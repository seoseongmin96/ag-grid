***** Import & Export / Multiple Sheets *****

< API Methods >

getSheetDataForExcel : 그리드의 데이터를 시트로 가져오는 데 사용되는 방법입니다. 나중에 getMultipleSheetAsExcel() 또는 exportMultipleSheetAsExcel() 에서 내보낼 수 있습니다.

exportMultipleSheetsAsExcel : 하나의 파일에 여러 시트의 Excel 내보내기를 다운로드합니다.

getMultipleSheetsAsExcel : exportMultipleSheetsAsExcel 와 유사하지만 파일을 다운로드하는 대신 사용자가 처리 할 Blob 을 반환합니다.
