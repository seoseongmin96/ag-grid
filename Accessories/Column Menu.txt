***** Accessories /Column Menu *****

< Customising the General Menu Tab >

getMainMenuItems : 기본 '열 머리글' 메뉴를 사용자 지정합니다.


< Repositioning the Popup >

postProcessPopup : 사용자가 팝업을 생성한 후 처리할 수 있도록 합니다. 응용 프로그램은 예를 들어 팝업의 위치를 변경하려는 경우 이를 사용할 수 있습니다.


< Customising the Columns Menu Tab >

suppressSyncLayoutWidthGrid : 그리드에서 재정렬될 때 열 레이아웃 업데이트를 억제하려면

suppressColumnFilter : 열 필터 섹션을 억제하려면

suppressColumnSelectAll : 모든 위젯 선택 / 선택 해제를 억제하려면

suppressColumnExpandAll : 모든 위젯 확장 / 축소를 억제하려면

contractColumnSelection : 기본적으로 열 그룹은 확장 시작됩니다. 계약된 그룹에 기본값을 true 전달
