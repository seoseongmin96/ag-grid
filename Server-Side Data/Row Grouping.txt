***** Server-Side Data / Row Grouping *****

< Configuring Group Levels >

getServerSideGroupLevelParams : 그룹화 수준에 따라 다른 매개변수를 제공할 수 있습니다.


< Debugging Group Levels >

getServerSideGroupLevelState : 모든 서버 측 그룹 수준에 대한 정보를 반환합니다.


< Open By Default >

isServerSideGroupOpenByDefault : 기본적으로 그룹을 열 수 있습니다.

getRoute : 행 노드의 경로를 반환합니다. 행 노드가 그룹이면 해당 행 노드로의 경로를 반환합니다. 행 노드가 그룹이 아닌 경우 undefined 를 반환합니다.


< Providing Child Counts >

getChildCount : 그룹 행의 하위 수를 설정할 수 있습니다.
