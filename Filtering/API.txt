***** Filtering / API *****

< Set Filter API >

getModel : 필터의 현재 상태를 나타내는 모델을 반환하거나 null 필터가 활성화되지 않은 경우 반환합니다.

setModel : 제공된 모델을 사용하여 필터의 상태를 설정합니다. null 일때 모델로 제공하면 필터가 비활성화됩니다.
           참고 : 필터 설정에 비동기적으로 값을 제공하는 경우 반환된 그리드 약속을 대기하여 추가 작업을 수행하기 전에 이러한 변경 사항이 적용될 때까지 기다려야 합니다.
           filter.setModel({ values : ['a','b']}).then(function(){
            gridApi.onFilterChanged(); });

getValues : 필터 설정에서 사용하는 고유 값의 전체 목록을 반환합니다.

setFilterValues : 즉석에서 필터 설정에 사용되는 값을 설정합니다.

refreshFilterValues : 원본 소스에서 필터에 표시된 값을 새로 고칩니다. 예를 들어 콜백이 제공된 경우 콜백이 다시 실행되고 필터가 반환된 값을 사용하여 새로 고쳐집니다.

resetFilterValues : 직접 제공된 값이 아닌 그리드의 값을 사용하도록 필터 설정을 재설정합니다.

getMiniFilter : 현재 미니 필터 텍스트를 반환합니다.

setMiniFiltr : 필터 상단의 미니 필터에 텍스트를 설정합니다.(팝업의 '빠른 검색').

isFilterActive : 필터가 현재 활성화되어 있으면 true를 반환하고 그렇지 않으면 false를 반환합니다. 활성화되면 1) 그리드가 열 머리글에 필터 아이콘을 표시하고 2) 필터가 데이터 필터링에 포함됩니다.

afterGuiAttached : 이 구성 요소에 대한 GUI가 화면에 렌더링된 직후에 필요한 작업을 수행하기 위한 후크입니다. 부모 팝업이 닫혔다가 다시 열리면 (예: 필터의 경우) 구성 요소가 표시될 때마다 이 메서드가 호출됩니다.
                   이는 특정 DOM 요소에 초점을 맞추는 것과 같이 실행 전에 첨부가 필요한 모든 논리에 유용합니다.
