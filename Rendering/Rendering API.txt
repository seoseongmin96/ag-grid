***** Rendering /Rendering API *****

< Displayed Rows API >

getDisplayedRowAtIndex : 지정된 위치에 표시된 RowNode 값을 반환합니다. index

getDisplayedRowCount : 표시된 행의 총 수를 반환합니다.

getFirstDisplayedRow : 스크롤로 인해 첫 번째로 표시된 행의 인덱스를 가져옵니다. (버퍼에 표시되지 않는 렌더링된 행 포함)

getLastDisplayedRow : 스크롤로 인해 마지막으로 표시된 행의 인덱스를 가져옵니다. (버퍼에 표시되지 않는 렌더링된 행 포함)
