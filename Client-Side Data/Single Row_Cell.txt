***** Client-Side Data /Single Row / Cell *****

< Single Row / Cell >

setData : rowNode의 데이터를 대체합니다. 완료되면 그리드가 표시되는 경우 렌더링된 전체 행을 새로 고칩니다.

setDataValue : rowNode 지정도니 열의 값을 바꿉니다. 완료되면 그리드는 필요한 행에서만 렌더링된 셀을 새로 고칩니다.
               colKey : 값이 업데이트되어야 하는 열
               newValue : 새 값
               eventSource : 이벤트의 소스
               반환값 : 값이 변경된 경우 True, 그렇지 않은 경우 False
