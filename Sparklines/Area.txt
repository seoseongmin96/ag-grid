***** Sparklines / AreaSparklineOptions *****

< AreaSparklineOptions >

type : 생성할 스파크라인의 유형입니다. 이 경우에는 'area' 입니다.

fill : 영역 채우기에 대한 CSS 색상 값입니다.
       default : 'rgba(124, 181, 236, 0.25)'

line : 회선 구성입니다.

marker : 표식 스타일에 대한 구성입니다.

crosshairs : 십자선의 구성입니다.

xKey : 데이터에서 X값을 검색하는 데 사용할 키입니다. 데이터 배열에 키-값 쌍이 있는 개체가 포함된 경우에만 사용됩니다.
       default : 'x'

yKey : 데이터에서 Y값을 검색하는 데 사용할 키입니다. 데이터 배열에 키-값 쌍이 있는 개체가 포함된 경우에만 사용됩니다. 
       default : 'y'

padding : 스파크라인 주변에 표시되는 패딩(픽셀) 구성입니다.

axis : 스파크라인의 축 선에 대한 옵션입니다.

highlightStyle : 항목 위에 마우스를 올렸을 때 사용되는 강조 표시 구성입니다.

tooptip : 툴팁 구성


< SparklineMarkerOptions >

enabled : 기본적으로 이것은 true로 설정되고 마커 크기는 0으로 설정됩니다. 이는 마커가 있지만 표시되지 않음을 의미합니다.
          default : true

shape : 마커의 모양입니다.
        default : 'circle'
        options : 'circle', 'diamond', 'square'

size : 마커의 너비(픽셀)입니다. 기본적으로 이것은 0이며 마커가 보이도록 크기를 눌립니다.

fill : 마커 채우기에 대한 CSS 색상 값입니다.
       default : 'rgb(124, 181, 236)'

shroke : 마커 외곽선의 CSS 색상 값입니다.
         default : 'rgb(124, 181, 236)'

strokeWidth : 마커 스트로크의 두께(픽셀 단위)입니다.
              default : 1

formatter : 개별 마커의 형식 스타일을 반환하는 콜백 함수입니다.
