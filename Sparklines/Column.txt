***** Sparklines / Columns *****

< ColumnSparklineOptions >

type : 생성할 스파크라인의 유형입니다. 이 경우에는 'column' 입니다.

fill : 열 채우기에 대한 CSS 색상 값입니다.
       default : 'rgb(124, 181, 236)'

stroke : 열 윤곽선의 CSS 색상 값입니다.

strokeWidth : 열의 획 두께(픽셀 단위)입니다.
              default : 0

paddingInner : 0과 1 사이의 비율로 나타낸 열 사이의 간격 크기입니다. 이 값은 밴드 시작과 다음 밴드 시작 사이의 간격인 "스텝"의 일부입니다.
               default : 0.1

paddingOuter :  첫 번째 열과 마지막 열의 바깥쪽, 즉 왼쪽과 오른쪽에 있는 패딩은 축을 위한 공간을 남겨두기 위한 것입니다. paddingInner와 관련하여 이 값은 0과 1 사이일 수 있습니다.
                default : 0.2

valueAxisDomain : 자동으로 결정된 도메인에 대한 사용자 재정의(데이터 최소값 및 최대값 기준). number축 에만 적용됩니다. 각 데이터 값에 해당하는 숫자 픽셀 값을 보간하는 데 사용됩니다.                                          

formatter : 개별 열이 나타내는 데이터를 기반으로 ColumnFormat 유형의 서식 스타일을 반환하는 콜백 함수입니다.

label : 레이블 구성.

xKey : 데이터에서 X 값을 검색하는 데 사용할 키입니다. 데이터 배열에 키-값 쌍이 있는 개체가 포함된 경우에만 사용됩니다.
       default : 'x'

yKey : 데이터에서 Y 값을 검색하는 데 사용할 키입니다. 데이터 배열에 키-값 쌍이 있는 개체가 포함된 경우에만 사용됩니다.
       default : 'y'

padding : 스파크라인 주변에 표시되는 패딩(픽셀) 구성입니다.

axis : 스파크라인 축 선에 대한 옵션입니다.

highlightStyle : 항목 위에 마우스를 올렸을 때 사용되는 강조 표시 구성입니다.

tooltip : 툴팁 구성.
