***** Components / Tooltip Component *****

< Tooltip Component Interface >

location : 응용 프로그램의 어떤 부분이 도구 설명을 표시하고 있습니까 (예: 'cell', 'header', 'menuItem' 등)

value : 도구 설명으로 렌더링할 값입니다.

valueFormatted : 도구 설명으로 렌더링할 서식이 지정된 값입니다.

colDef : 열 / 열 그룹 정의

column : 열 / 열 그룹

rowIndex : 도구 설명을 렌더링하는 셀이 포함된 행의 인덱스입니다.

node : 행 노드입니다.

data : 문제의 행 노드에 대한 데이터입니다.

api : 그리드 API

columnApi : 열 API

context : gridOptions.context 에 설정된 애플리케이션 컨텍스트입니다.
