***** Standalone Charts / Bar & Column *****

< API Reference >

xKey : 데이터에서 x 값을 검색하는 데 사용할 키입니다.

type : '바' | '열'

grouped : 서로 다른 y 값을 별도의 막대로 표시할지 (그룹화) 또는 표시하지 않을지 (누적) 여부입니다.
          default : false

stacked : 부울

normalizedTo : 막대 스택을 정규화할 숫자입니다. true 일 때는 효과가 grouped 없습니다. 예를 들어 normalizedTo를 100으로 설정하면 각 막대 스택의 총합이 100이 되도록 막대 스택의 크기가 모두 비례적으로 조정됩니다.

yKey : 데이터에서 y 값을 검색하는 데 사용할 키입니다.

xName : x 값에 대한 사람이 읽을 수 있는 설명입니다. 제공된 경우 기본 툴팁에 표시되고 툴팁 렌더러에 매개변수 중 하나로 전달됩니다.

yName : y 값에 대한 사람이 읽을 수 있는 설명입니다. 제공된 경우 기본 툴팁에 표시되고 툴팁 렌더러에 매개변수 중 하나로 전달됩니다.

flipXY : 부울

fill : 영역 채우기에 사용할 색상입니다.

stroke : 막대의 획에 사용할 색상입니다.

strokeWidth : 막대의 획 너비(픽셀 단위)입니다.
              default : 1

fillOpacity : 막대 채우기의 불투명도입니다.
               default : 1

strokeOpacity : 막대의 획 불투명도입니다.
                default : 1

lineDash :  막대/열 스트로크가 렌더링되는 방식을 정의합니다. 배열의 모든 숫자는 대시와 간격이 번갈아 나타나는 길이(픽셀 단위)를 지정합니다. 
            예를 들어, 는 픽셀 사이에 간격이 있는 픽셀 [6, 3]길이의 대시를 의미 합니다.

lineDashOffset : 점선의 초기 오프셋(픽셀)입니다.
                 default : 0

shadow : 차트 게열 뒤에 사용되는 그림자에 대한 구성입니다.

label : 막대에 표시된 레이블에 대한 구성입니다.

tooltip : 시리즈별 툴팁 구성.

formatter : 지정된 매개변수를 기반으로 개별 막대/열의 형식을 반환하는 데 사용되는 함수입니다. 현재 막대/열이 강조 표시되면 highlighted 속성이 true로 설정됩니다. 
            강조 표시된 상태와 강조 표시되지 않은 상태를 구분하려면 이 항목을 확인하십시오.

listeners : 이벤트 리스너에 대한 이벤트 이름의 맵

id : 시리즈의 기본 식별자입니다. 이는 seriesId여러 계열을 구별하기 위해 사용자 콜백에서와 같이 제공됩니다. 자동 생성된 ID는 향후 경고 없이 변경될 수 있습니다. 
     콜백이 시리즈별로 동작을 변경해야 하는 경우 고유한 id값을 제공하세요.
     default : 자동 생성 값

data : 시리즈를 렌더링할 때 사용할 데이터입니다. 이것이 제공되지 않으면 대신 차트에 데이터를 설정해야 합니다.

visible : 시리즈 표시 여부입니다.

showInLegend : 시리즈를 범례에 포함할지 여부입니다.

cursor : 가리킨 영역 마커에 사용할 커서입니다. 이 구성은 CSS cursor 속성과 동일합니다.

highlightStlye : 마커 / 데이터 포인트 또는 범례 항목 위에 마우스를 올렸을 때 시리즈 마커 및 시리즈 라인 하이라이트 구성
