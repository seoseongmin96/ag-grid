***** Standalone Charts / Axes *****

< API Reference >

position : 축을 렌더링해야 하는 차트의 위치입니다.

title : 축 옆에 표시되는 제목에 대한 구성입니다.

line : 축 선에 대한 구성입니다.

label : 눈금 옆에 표시되는 축 레이블에 대한 구성입니다.

gridStyle : 차트 영역에서 그리드를 형성하는 데 사용되는 선 구성

crossLines : 데이터 값에 해당하는 교차선 또는 영역을 추가합니다.

keys : 끈 []

thickness : 0이 아닌 값으로 설정하면 레이블 크기에 관계없이 축이 지정된 두께를 갖습니다.
