***** Standalone Charts / Axes *****

< API Reference >

position : 축을 렌더링해야 하는 차트의 위치입니다.

title : 축 옆에 표시되는 제목에 대한 구성입니다.

line : 축 선에 대한 구성입니다.

label : 눈금 옆에 표시되는 축 레이블에 대한 구성입니다.

gridStyle : 차트 영역에서 그리드를 형성하는 데 사용되는 선 구성

crossLines : 데이터 값에 해당하는 교차선 또는 영역을 추가합니다.

keys : 끈 []

thickness : 0이 아닌 값으로 설정하면 레이블 크기에 관계없이 축이 지정된 두께를 갖습니다.


< Category Axis >

type : '범주'

paddingInner : 0과 1사이의 비율로 범주 간 간격의 크기입니다. 이 값은 밴드 시작과 다음 밴드 시작 사이의 간격인 '스텝' 의 일부입니다.
               default : 0.2

paddingOuter : 첫 번째와 마지막 범주의 바깥쪽, 즉 왼쪽과 오른쪽의 패딩입니다. paddingInner 와 관련하여 이 값은 0과 1사이일 수 있습니다.
               default : 0.3

groupPaddingInner : 이 속성은 범주 축에 그려진 그룹화된 열/막대 계열에 대한 것입니다. 축을 따라 단일 그룹 내 막대 또는 기둥 사이의 간격 크기를 결정하는 0과 1 사이의 비율입니다.
                    default : 0.2

tick : 축 틱에 대한 구성입니다.

crossLines : 데이터 값에 해당하는 교차선 또는 영역을 추가합니다.

thickness : 0이 아닌 값으로 설정하면 레이블 크기에 관계없이 축이 지정된 두께를 갖습니다.


< Number Axis >

type : '숫자'

nice : 'true' 이면 범위가 반올림되어 눈금 사이의 균등한 간격을 보장합니다.
        default : true

min : 자동으로 결정된 최소값에 대한 사용자 재정의(계열 데이터 기반).

max : 자동으로 결정된 최대값에 대한 사용자 재정의(시리즈 데이터 기반).

tick : 축 틱에 대한 구성입니다.

corssLines : 데이터 값에 해당하는 교차선 또는 영역을 추가합니다.

thickness : 0이 아닌 값으로 설정하면 레이블 크기에 관계없이 축이 지정된 두께를 갖습니다.


< Log Axis >

type : '통나무'

nice : 'true' 이면 범위가 반올림되어 눈금 사이의 균등한 간격을 보장합니다.
        default : true

min : 자동으로 결정된 최소값에 대한 사용자 재정의(계열 데이터 기반).

max : 자동으로 결정된 최대값에 대한 사용자 재정의(시리즈 데이터 기반).

base : 사용자 로그의 밑입니다.
       default : 10

tick : 축 틱에 대한 구성입니다.

crossLines : 데이터 값에 해당하는 교차선 또는 영역을 추가합니다.

thickness : 0이 아닌 값으로 설정하면 레이블 크기에 관계없이 축이 지정된 두께를 갖습니다.
